# TutoVera perfected exact work bundle

This bundle uses the approved generated work from the last visual pass. The production PNGs are cropped from those approved sheets; logos/wordmarks/lockups were cleaned only to make the background genuinely transparent where appropriate. The source artwork itself was not regenerated or redrawn.

Folders:
- `public/brand/exact/` contains the main production-ready assets.
- `public/brand/dark-mode/` contains the flipped app icon plus the light-letter dark-mode logo/wordmark/lockup versions.
- `public/brand/transparent/` contains explicit transparent-name copies for easy imports.
- `public/brand/source-generated-exact-sheets/` contains the unchanged approved generated sheets.
- `public/brand/previews/` contains a contact sheet to visually verify the extraction.

Note on SVG files: because the approved work is raster artwork, the SVG files embed the exact PNG assets rather than re-vectorizing them. This prevents changes to shape, spacing, gradients, and typography.
